# AWS Serverless App

This is a simple app triggerring CRUD events working with AWS DynamoDB through AWS API Gateway

&nbsp;
&nbsp;

**Used Stacks:**
- ReactJS (13.4.0)
- AWS Lambda (Python 3.11)
- AWS DynamoDB
- AWS API Gateway

&nbsp;
&nbsp;

**Installed Node Packages (Dependecies)**
- axios@1.5.1
- cors@2.8.5
- html-to-react@1.7.0
- jquery@3.7.1
- react-dom@18.2.0
- react-scripts@5.0.1
- react@18.2.0
- web-vitals@2.1.4
